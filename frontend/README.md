# RiskShield-Gig Frontend 🛡️

This is the worker-facing web application for RiskShield-Gig, built using **Next.js** and **TypeScript**. It provides a low-friction interface for delivery workers to register, subscribe to coverage, and track automated claim payouts.

## ✨ Premium UI & Aesthetic
- **Modern Design**: Built with glassmorphism, fluid gradients, and a dark-mode-first aesthetic.
- **Micro-Animations**: Uses CSS and Tailwind transitions for interactive risk scores and status updates.
- **Responsive**: Fully optimized for mobile browsers (PWA-ready).

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS
- **Typing**: TypeScript
- **State**: React Hooks (useState, useEffect)

## 📂 Key Folders
- `src/app/`: Next.js pages and layouts.
- `src/components/`: Reusable UI components.
- `public/`: Static assets (SVG, images).

## 🚀 Getting Started

### Prerequisites
- Node.js (v18.17+ recommended)
- npm or yarn

### Installation
```bash
cd frontend
npm install
```

### Running the Development Server
```bash
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## 📊 Key Features
- **Smart Registration**: Onboard workers with localized zone selection and instant AI-based risk profiling.
- **Coverage Dashboard**: (In Progress) View active policies and claim statuses.
- **Instant Visualization**: Real-time feedback on risk levels and weekly premiums.
